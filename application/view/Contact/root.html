How to contact us?

There are several ways to put yourself in touch with us

GitHub

Mamba is a Free Software project, so you can redistribute it and/or modify it under the terms of the GNU General Public License version 3 or any later version. The Mamba's source code is public and available in both Github and Bitbucket git repositories.

Twitter

You can follow us in our official Twitter account. We use this account to inform about the latests news related with Mamba as releases, bug fixes, security issues and more. We also try to reply to anyone that put in contact with us through Twitter. Just follow the @pymamba account.

EMail

You can also get involve registering yourself in any of the following mailing lists:

  • Support Google Groups - mamba support list
  • Learn Google Groups - mamba learning center list
  • Developers Google Groups - core-developers list
