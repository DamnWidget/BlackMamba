Introducing Mamba Framework Components

Twisted Framework

Mamba is built on top of Twisted Web and uses Twisted as WSGI web server as well. Twisted is an event-driven newtork programming framework completely written in Python and licensed under the MIT License.

Storm ORM

Mamba uses Storm as Object Relational Mapper. Storm is and object-relational mapper for Python developed at Canonical. Storm is successfully used in Canonical projects such as Launchpad and Landscape. Storm was released on July 9th, 2007 under the GNU LGPL License

Jinja2

While Twisted has its own templating system, Mamba uses Jinja2 as primary templating system. Jinja2 is a full featured template engine for Python with full unicode support, and integrated sandboxed execution environment licensed with the BSD 3-Clause License.
